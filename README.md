# Static Website Hosting on Amazon S3

This project demonstrates how to host a static HTML/CSS/JavaScript website using Amazon S3 with public access enabled.

---

## 📌 Project Objective
Host a static website on Amazon S3 and make it publicly accessible.  
The project also covers the concept of custom domain integration at a theoretical level.

---

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- Amazon S3

---

## 🚀 Implementation Details

### 1. Amazon S3 Bucket Setup
- Created an S3 bucket
- Uploaded static website files (HTML, CSS, JS)
- Enabled public read access using bucket policy

### 2. Static Website Hosting
- Enabled **Static Website Hosting** in S3 bucket properties
- Configured `index.html` as the entry point

### 3. Public Access
- Configured bucket policy to allow public access for website files
- Verified website accessibility via browser
